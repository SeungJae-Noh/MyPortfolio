# PowerPoint 포트폴리오 작성 가이드

## 📋 개요

이 문서는 노승재 백엔드 개발자 포트폴리오를 PowerPoint로 작성하기 위한 상세 가이드입니다. 각 슬라이드별 레이아웃, 디자인, 내용 배치를 구체적으로 제안합니다.

## 🎨 디자인 가이드라인

### 색상 팔레트

#### 주 색상
- **다크 블루그레이**: `#2C3E50`
  - 제목, 강조 텍스트, 헤더 배경
  - RGB: (44, 62, 80)

#### 강조 색상
- **블루**: `#3498DB`
  - 링크, 아이콘, 강조 요소
  - RGB: (52, 152, 219)

#### 보조 색상
- **라이트 그레이**: `#ECF0F1`
  - 배경, 구분선
  - RGB: (236, 240, 241)

#### 텍스트 색상
- **다크 그레이**: `#34495E`
  - 본문 텍스트
  - RGB: (52, 73, 94)

#### 강조 텍스트
- **화이트**: `#FFFFFF`
  - 다크 배경 위 텍스트
  - RGB: (255, 255, 255)

### 폰트

#### 제목 폰트
- **Noto Sans KR Bold** 또는 **Pretendard Bold**
- 크기: 32-44pt
- 제목 슬라이드: 48-60pt

#### 본문 폰트
- **Noto Sans KR Regular** 또는 **Pretendard Regular**
- 크기: 18-24pt
- 서브텍스트: 14-16pt

#### 코드/기술 스택
- **Consolas** 또는 **Fira Code**
- 크기: 14-18pt

### 간격 및 레이아웃

- **슬라이드 여백**: 상하좌우 0.5인치 (1.27cm)
- **제목과 본문 간격**: 0.3인치 (0.76cm)
- **Bullet Point 간격**: 0.2인치 (0.51cm)
- **섹션 간 간격**: 0.4인치 (1.02cm)

## 🎯 PowerPoint 작성 팁

### 1. 일관성 유지
- 모든 슬라이드에서 동일한 폰트, 색상, 스타일 사용
- 제목 슬라이드는 항상 동일한 위치와 스타일
- Bullet Point 스타일 통일

### 2. 시각적 계층 구조
- 제목 > 부제목 > 본문 순으로 명확한 계층 구조
- 중요한 정보는 큰 폰트, 볼드, 색상으로 강조
- 숫자와 성과는 특히 눈에 띄게 표현

### 3. 여백 활용
- 슬라이드를 꽉 채우지 말고 적절한 여백 유지
- 내용이 많으면 슬라이드를 나누기
- 한 슬라이드당 최대 5-6개 bullet point

### 4. 아이콘 및 이미지
- 이모지나 아이콘을 적절히 활용 (과하지 않게)
- 기술 스택은 로고 이미지 사용 가능
- 다이어그램은 간단하고 명확하게

### 5. 애니메이션 (선택사항)
- 과도한 애니메이션은 피하기
- 필요시 페이드 인/아웃만 사용
- 슬라이드 전환은 부드럽게

### 6. 프린트 고려
- 프린트 시에도 가독성 유지
- 색상 대비 확인
- 폰트 크기 충분히 크게
