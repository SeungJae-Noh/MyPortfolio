## 🚀 주요 프로젝트

### 1. BrainStims 멀티테넌트 서버 시스템 (2022.01 ~ 현재)

#### 프로젝트 개요
뇌 자극 치료 시스템을 위한 멀티테넌트 백엔드 서버 설계 및 개발. Database-per-Tenant 패턴으로 각 병원별 독립 데이터베이스 운영.

#### 담당 역할
- 백엔드 아키텍처 설계 및 개발
- RESTful API 설계 및 구현 (20+ 엔드포인트)
- JWT 기반 인증/인가 시스템 구축
- AWS 클라우드 인프라 구축 및 운영

#### 주요 기술 성과

##### 1️⃣ 멀티테넌트 아키텍처 구현
- **Database-per-Tenant 패턴**으로 병원별 데이터 완전 격리
- 동적 DB 세션 관리로 요청당 적절한 테넌트 DB 자동 라우팅
- 엔진 캐싱 메커니즘으로 DB 커넥션 재사용 및 성능 최적화

##### 2️⃣ 인증/보안 시스템
- **JWT 기반 Access Token (15분) / Refresh Token (7일)** 체계 구축
- **Redis Cluster** 활용한 토큰 블랙리스트 관리
- **AWS Secrets Manager** 연동으로 민감 정보 안전 관리

##### 3️⃣ 인프라 구축
- **AWS 클라우드 환경 설계**: EC2, RDS, ALB, ElastiCache, Route53, ACM
- **Gunicorn + Nginx** 조합으로 고가용성 웹 서버 구성
- **Docker 컨테이너화**로 개발/스테이징/프로덕션 환경 통일

##### 4️⃣ 모니터링 및 운영
- DB 연결 현황 모니터링 API 구현
- 테넌트별 헬스 체크 시스템 구축
- 로그 로테이션 및 에러 추적 시스템 구축

#### 기술 스택
- **Backend**: Flask 3.1.0, SQLAlchemy 2.0.40, Flask-JWT-Extended 4.7.1
- **Database**: MariaDB, Redis Cluster
- **Infrastructure**: AWS, Docker, Nginx, Gunicorn

#### 프로젝트 성과
- ✅ **5개 이상의 병원 테넌트** 안정적 운영 중
- ✅ **API 응답 시간 평균 100ms 이하** 유지
- ✅ **99.9% 서버 가동률** 달성
